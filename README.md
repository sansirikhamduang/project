# 🚗 License Plate Detection System

## 📌 รายละเอียดโปรเจค
โปรเจคนี้เป็นระบบตรวจจับป้ายทะเบียนรถโดยใช้เทคโนโลยี Computer Vision และ Image Processing 
เพื่อระบุและอ่านข้อความจากป้ายทะเบียนรถอัตโนมัติ

## 🎯 วัตถุประสงค์
- จัดเก็บและจัดการข้อมูลทะเบียนรถของผู้อยู่อาศัยในโครงการ
- อำนวยความสะดวกในการค้นหาและตรวจสอบข้อมูลทะเบียนรถ
- เพิ่มประสิทธิภาพในการดูแลความปลอดภัยและการเข้า-ออกของรถยนต์

## 🛠️ เทคโนโลยีที่ใช้
- PHP (สำหรับพัฒนาเว็บไซต์)
- MySQL (ฐานข้อมูล)
- HTML, CSS (สำหรับแสดงผลหน้าเว็บ)
- Tesseract OCR (สำหรับอ่านป้ายทะเบียน)

## ⚙️ วิธีการทำงาน
1. ผู้ดูแลหรือเจ้าหน้าที่บันทึกข้อมูลทะเบียนรถและข้อมูลผู้อยู่อาศัยผ่านหน้าเว็บ
2. ระบบจัดเก็บข้อมูลลงในฐานข้อมูล MySQL
3. สามารถค้นหา แก้ไข หรือลบข้อมูลทะเบียนรถได้จากหน้าเว็บ
4. แสดงรายการทะเบียนรถทั้งหมด หรือค้นหาเฉพาะรายการที่ต้องการ
5. ระบบแจ้งเตือนหรือบันทึกประวัติการเข้า-ออกของรถยนต์

## ⚙️ วิธีติดตั้งระบบ

1. ติดตั้ง XAMPP หรือโปรแกรมจำลองเซิร์ฟเวอร์
2. เปิด Apache และ MySQL
3. นำโฟลเดอร์ project ไปไว้ในโฟลเดอร์ htdocs
4. เข้า phpMyAdmin เพื่อสร้างฐานข้อมูล
5. Import ไฟล์ฐานข้อมูล
6. เปิดเว็บผ่าน:
   [เปิดระบบ](http://localhost/project)

## ▶️ วิธีใช้งาน
เข้าสู่ระบบด้วยบัญชีผู้ดูแลหรือเจ้าหน้าที่
ไปที่เมนู "จัดการทะเบียนรถ" หรือ "ผู้อยู่อาศัย"
เพิ่มข้อมูลทะเบียนรถและข้อมูลผู้อยู่อาศัยผ่านแบบฟอร์มในหน้าเว็บ
สามารถค้นหา แก้ไข หรือลบข้อมูลทะเบียนรถได้ทันที
ระบบจะแสดงรายการทะเบียนรถทั้งหมด หรือค้นหาเฉพาะรายการที่ต้องการ

## 📌 ตัวอย่างการใช้งาน
- ระบบลานจอดรถ
- ระบบหมู่บ้าน / คอนโด
- ระบบตรวจสอบรถเข้า-ออก

## 📸 ตัวอย่างผลลัพธ์
Input: ข้อมูลทะเบียนรถ เช่น "กข 1234"
Output: ข้อมูลเจ้าของรถและรายละเอียดที่เกี่ยวข้อง
